npm install -g typescript   ** Instalo Typescript de forma global
tsc --version               ** Chequeo la version instalada

npm i typescript            ** instalamos typescript en el proyecto
Creamos el tsconig.json     ** Contiene toda la informacion de configuracion de typescript
